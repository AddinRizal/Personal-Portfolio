About Me

Saya adalah mahasiswa jurusan Informatika di Universitas Sebelas Maret yang penuh semangat dan berkomitmen untuk mengembangkan keterampilan dalam dunia teknologi. Selain kesukaan saya pada coding dan pemrograman, saya juga menikmati berorganisasi dan bekerja dalam kelompok, karena saya percaya bahwa kerjasama tim memperkaya pengalaman dan hasil akhir proyek.

Dengan landasan pendidikan di bidang Informatika, saya memiliki keinginan kuat untuk menjadi seorang full stack web developer. Saya melihat coding bukan hanya sebagai keterampilan teknis, tetapi juga sebagai sarana untuk menciptakan solusi inovatif yang dapat memberikan dampak positif.

Pada saat ini, saya aktif mengejar cita-cita saya dengan menggabungkan pengetahuan akademis saya dengan pengalaman praktis dalam mengembangkan berbagai proyek web. Saya percaya bahwa kombinasi antara keterampilan teknis dan kemampuan berorganisasi akan membantu saya mencapai tujuan sebagai seorang pengembang web full stack yang sukses.
